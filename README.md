```markdown
# 🎮 Guess My Number - Streamlit Game

A simple web-based number guessing game built using **Python** and **Streamlit**.

The computer randomly generates a number between **1 and 100**, and the player tries to guess it. The app provides hints like **Too High** or **Too Low** until the correct number is guessed.

---

## 🚀 Features

- Random number generation
- User guess input
- Hint system (Too High / Too Low)
- Attempt counter
- Restart game functionality
- Interactive web UI using Streamlit

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Random Module

---

## 📂 Project Structure

```

guess-number-streamlit/
│
└── app.py

````

---

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Manoj17tech/Python-Project.git
````

Move into the project folder:

```bash
cd guess-number-streamlit
```

Install dependencies:

```bash
pip install streamlit
```

---

## ▶️ Run the Application

```bash
streamlit run app.py
```

Open your browser and go to:

```
http://localhost:8501
```

---

## 🎯 How to Play

1. The computer selects a random number between **1 and 100**.
2. Enter your guess in the input box.
3. Click **Submit Guess**.
4. The app will tell you:

   * **Too High**
   * **Too Low**
   * **Correct Guess**
5. Continue until you find the correct number.

---

